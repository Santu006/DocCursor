# Sample Law Firm Matter — Client-ABC-Corp

Synthetic but realistic client folder for DocCursor / Law-firm Q&A demos.

**Not real client data.** All parties, case numbers, and amounts are fictional.

## Structure

```
Client-ABC-Corp/
  Contracts/
    distributor_agreement.txt
    nda_2024.txt
    supply_contract.txt
  Litigation/
    case_2023_petition.txt
  Corporate/
    board_minutes.txt
  Correspondence/
    client_emails.txt
```

## How to use in DocCursor

1. Open workspace (or create `Client-ABC-Corp`).
2. Use **Upload folder** and select `Client-ABC-Corp`.
3. Ask matter questions with documents selected (or workspace-wide reverse lookup).

## Suggested test questions

- Who are the parties and territory in the distributor agreement?
- What is the NDA term and survival period?
- Compare LC / payment security in distributor vs supply contracts.
- What relief is sought in the 2023 petition?
- What did the board authorize on March 12, 2024?
- which agreement has letter of credit 250000
- which agreement has monthly fee not applicable — use: which document mentions liquidated damages 50000
